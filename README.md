# 🚀 Modern ATS CV Analyzer
> Yapay zeka destekli, veri odaklı ve modern İnsan Kaynakları için Akıllı Aday Takip Sistemi (ATS) Analizörü.

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)

Modern iş dünyasında binlerce CV arasından doğru adayı bulmak samanlıkta iğne aramak gibidir. **Modern ATS CV Analyzer**, karmaşık özgeçmiş verilerini gelişmiş NLP (Doğal Dil İşleme) ve veri bilimi teknikleriyle işleyerek, adayların iş ilanlarıyla olan uyumunu saniyeler içinde skorlar ve görselleştirir.

## ✨ Öne Çıkan Özellikler

* **🔍 Akıllı Anahtar Kelime Eşleştirme:** İş ilanı gereksinimleri ile CV içeriğini semantik olarak karşılaştırır.
* **📊 Görsel Analitik Dashboard:** Aday skorlarını, yetkinlik dağılımlarını ve eksik becerileri interaktif grafiklerle sunar.
* **⚡ Hızlı İşleme (Pipeline):** Yüzlerce PDF/Word formatındaki özgeçmişi saniyeler içinde yapılandırılmış veriye dönüştürür.
* **🎯 Tarafsız Değerlendirme Modeli:** Formattan bağımsız, tamamen veriye dayalı objektif bir sıralama algoritması kullanır.
